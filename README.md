# InfinityOS

InfinityOS is a modern desktop environment built with Python and GTK4.

Its goal is to create a minimal, context-aware desktop experience that adapts to the user's current task.

## Current Status

Genesis v0.1

## Features

- GTK4 application
- Component architecture
- Live clock
- Context engine foundation
- Interactive Infinity button

## Roadmap

- Infinity Menu
- Floating Bar
- Focus Mode
- Dashboard
- AI Context Engine

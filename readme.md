#InfinityOS

InfinityOS is a modern, dynamic desktop environment for Linux built around personalization, performance, and automation.

Philosophy

InfinityOS is a desktop that adapts to how you work, not the other way around.

## Features

- 🎨 Dynamic Pywal colors
- 🖼 Weather-aware wallpapers
- 🎵 Centered media controls
- 🌤 Live weather
- 🌡 CPU temperature
- 🧠 Memory monitor
- 📅 Calendar
- 🕒 Clock
- ✨ Smooth animations
- 🚀 Developer-first workflow
